使用Dashboard

Dashboard为起始页，您可以浏览、创建、导入项目。

界面总览

打开项目

选择标签栏的打开项目

新建项目

选择标签栏的新建项目，并选择一个项目模板及路径后，点击下方的新建项目按钮。

帮助

选择标签栏的帮助，帮助能使您对cocos creator有一个初步的了解。
