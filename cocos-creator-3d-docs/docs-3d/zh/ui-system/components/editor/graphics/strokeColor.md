# strokeColor

`strokeColor` 属性设置或返回用于笔触的颜色。


## 实例

```javascript
var ctx = node.getComponent(cc.GraphicsComponent);
ctx.lineWidth = 2;
ctx.strokeColor = cc.hexToColor('#0000ff');
ctx.rect(20,20,250,200);
ctx.stroke();
```
