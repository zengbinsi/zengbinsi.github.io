# UI 变换组件

Cocos Creator 3D 大多数的节点变换操作已经在节点在节点上完成，对于 UI 部分由于涉及到 UI 尺寸以及 UI 锚点问题，所以衍生出了 UI 变换组件。一般用于点击事件的计算，以及适配的作用。

## UITransform 属性介绍

| 属性 |   功能说明
| -------------- | ----------- |
| ContentSize | UI 内容尺寸
| AnchorPoint | UI 锚点位置
