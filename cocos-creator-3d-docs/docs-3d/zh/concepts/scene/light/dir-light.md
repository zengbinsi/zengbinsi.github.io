# 主方向光（Main Directional Light）

Cocos Creator 3D 中只有一个主方向光，主方向光可以理解为场景中的主导性光源，通常是室外场景的太阳光。主方向光还会影响阴影的投影。

![main dir light](dir-light.jpg)

| 参数名称 | 说明 |
|:-------:|:---:|
| Color | 光源颜色 |
| UseColorTemperature | 是否启用色温 |
| ColorTemperature | 色温 |
| Illumiance | 照度，单位**勒克斯（*lx*）** |
