Class Rect

轴对齐矩形。 矩形内的所有点都大于等于矩形的最小点 (xMin, yMin) 并且小于等于矩形的最大点 (xMax, yMax)。 矩形的宽度定义为 xMax - xMin；高度定义为 yMax - yMin。
