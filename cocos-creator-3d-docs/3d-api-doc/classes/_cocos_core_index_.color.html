Class Color

通过 Red、Green、Blue 颜色通道表示颜色，并通过 Alpha 通道表示不透明度。 每个通道都为取值范围 [0, 255] 的整数。

toHEX

  • toHEX(fmt: "#rrggbb" | "#rrggbbaa"): string
    • Defined in cocos/core/value-types/color.ts:298

    转换当前颜色为十六进制颜色字符串。

    example

    const color = new Color(255, 14, 0, 255); color.toHex("rrggbbaa"); // "FF0E00FF" color.toHex("rrggbb"); // "FF0E00"

    Parameters

    • fmt: "#rrggbb" | "#rrggbbaa"

      格式选项。

      • '#rrggbbaa' 获取Red、Green、Blue、Alpha通道的十六进制值（两位，高位补 0）并依次连接；
      • '#rrggbb 与 '#rrggbbaa' 类似但不包括 Alpha 通道。

    Returns string

    十六进制颜色字符串。
